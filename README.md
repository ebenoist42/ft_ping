# ft_ping — créé par ebenoist

Recodage de la commande `ping` (projet École 42).

## Référence : inetutils-2.0

Le sujet demande de reproduire le comportement de la commande `ping`
de **inetutils-2.0**.

Grâce à la VM, j'ai pu compiler cette version de référence.
Voici la commande pour obtenir le comportement de inetutils-2.0 :

```bash
sudo ~/inetutils-2.0/ping/ping google.com
```
